csharp 临时让出 sqlite 数据库

使用 .Dump

SQLite 提供了 .dump 命令，可以将整个数据库导出为 SQL 脚本。你可以通过执行此命令生成备份文件，然后传输该文件。

实现步骤

  1. 使用 SQLite 命令行工具或代码执行 .dump 命令。
  2. 将生成的 SQL 文件传输到前端。

使用 VACUUM（推荐）

SQLite 3.27.0 及以上版本支持 VACUUM INTO 命令，可以将数据库备份到一个新的文件中。

实现步骤

  1. 执行 VACUUM INTO 命令，将数据库备份到新文件。
  2. 传输新文件。
