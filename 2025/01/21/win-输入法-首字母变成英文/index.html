win 输入法 首字母变成英文
发表于 2025-01-21 | 更新于 2025-01-21
windows
windows 输入法

输入法问题

现象：在切换窗口或输入框后，接着开始打字，会出现拼音首字母直接出现在输入框内，按下第二个字母才出现输入法。

例如：切换窗口后敲击键盘nihc，期待出现的是打出你好的输入法界面，然而n字母直接出现在输入框内，随后输入法出现，拼音为ihc，打出长存。

解决办法

  1. 使用 Win + i 打开设置界面。
  2. 转到个性化，任务栏
  3. 下滑找到触摸键盘改成始终
  4. 桌面右下角任务栏出现键盘图标，点击打开屏幕键盘一次，再关闭，问题解决。
